# Christian.FM

A beautiful Christian music radio and streaming app. Built with React, Vite, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Live radio stations for Worship, Gospel, Contemporary, Hymns, and more
- Curated playlists for every moment of your day
- Beautiful, modern dark-themed UI
- Full music player with playback controls, shuffle, repeat, and volume
- Collapsible sidebar navigation
- Favorites and listening history
- Daily scripture verse

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4
- Lucide Icons
