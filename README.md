# Christian.FM

Christian.FM is a clean, modern radio-style web app UI for faith-based music.

## Current UI Includes

- Branded hero section with clear Christian.FM identity
- Interactive demo "Play Stream" button
- Now Playing card with progress animation and live clock
- Featured stations, schedule, and queue sections
- Responsive design for desktop, tablet, and mobile

## Run Locally

Because this first version is a lightweight static app, you can open it quickly:

1. Open `index.html` directly in your browser
2. Or serve it from a local web server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.
